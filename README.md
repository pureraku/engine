# OpenGL Playground

Small modern OpenGL playground with a simple renderer-style architecture.

Built with:

- **Rust** and **Cargo**
- **glfw** — window + OpenGL context
- **glow** — OpenGL bindings
- **glam** — math types
- **image** — texture loading

---

## Build

```bash
cargo build
```

## Run

```bash
cargo run
```
